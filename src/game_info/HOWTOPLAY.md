## How to Play

### Objective
Explore the rooms, find the key, and unlock the exit to win the game.

---

### Controls

Type commands into the console:
search → look for items in the current room
take <item> → pick up an item (example: take key)
move <direction> → move between rooms (example: move north)
use <item> → use an item (example: use key)
quit → exit the game


---

### Example Gameplay
search
take key
move north
move east
use key


---

### Win Condition

- The player must:
    - Find and take the **Key**
    - Navigate to the **Exit room**
    - Use the key with `use key`

---

### Notes

- Some actions only work in specific rooms
- Invalid commands will prompt an error message
- The game is text-based and runs in the console
